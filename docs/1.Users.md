# Users Endpoint
1. Create User
```
curl -X POST -H "Content-type: application/json" http://localhost:3001/users/register -d '{"first_name": "Juan", "last_name": "dela Cruz", "password": "MySupers3cretPassword", "username":"jdelacruz", "email": "jdelacruz@geospectrum.xyz", "region": "central"}'
```
2. Login
```
curl -X POST -H "Content-type: application/json" http://localhost:3001/users/login -d '{"password": "MySupers3cretPassword", "username":"jdelacruz"}'
```
